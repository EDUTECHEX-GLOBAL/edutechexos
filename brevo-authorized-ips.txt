Brevo email is sent via SMTP relay (smtp-relay.brevo.com:587).
SMTP has NO IP allowlist — Brevo HTTP API IP restrictions do not apply.

Required env vars (Render + Vercel):
  SMTP_HOST=smtp-relay.brevo.com
  SMTP_PORT=587
  SMTP_USER=<your Brevo SMTP login>
  SMTP_PASS=<your Brevo SMTP key>
  SMTP_FROM=EduTechExOS <noreply@edutechex.in>
